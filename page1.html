Getting Started with SWAPI

In this module we're going to go over how to get started with the SWAPI.
You'll Learn:

  • SWAPI basics
  • Getting info from the call

SWAPI Basics

The SWAPI is a database with all different kinds of resources from the Star Wars universe.
There are 6 main resources we can tap into:

  • People: 87
  • Films: 7
  • Starships: 37
  • Vehicles: 39
  • Species: 37
  • Planets: 61
Each of these resources has various attributes we can access as well as different ways we can access them. The SWAPI is completely open so there is no need for an API key, however, it is rate limited to 10,000 API request per day per IP.
To make a request all we need is the base URL:
http://swapi.co/api/
From there we can choose which resource we want to tap into.

Getting Info From The Call

If we make a simple call to the SWAPI we can use the JSON response to get information from that call. Lets use the Planets resource as an example since that's what we'll be working with.
If we make a simple call to:
http http://swapi.co/api/planets/
When we console.log() the response we will see something like this:

As we can see the call to http http://swapi.co/api/planets/ gives us a lot of information. It tells us the count of planets 61. It tells us the base URLs of the next and previous pages of planets and it gives us an array of 10 planet objects. In this call I expanded the planet object at index 0 so we can see what attributes we can get from each planet. We can access its climate, films it's featured in, its name, its residents and much more information.
